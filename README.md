# Viva Food - Catálogo

Um catálogo interativo de comidas para visualização em slideshow.

## Recursos

- 📸 Visualização de slides automática
- ⏯️ Controles de reprodução (Play/Pausa)
- ⌨️ Navegação com teclado (setas e barra de espaço)
- 📱 Responsivo para desktop e mobile
- 🎨 Design elegante com tema escuro

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador web
3. Use os botões ou teclado para navegar

## Navegação

- **Anterior**: botão ◀ ou seta esquerda
- **Seguinte**: botão ▶ ou seta direita  
- **Pausa/Play**: botão ⏸ ou barra de espaço
